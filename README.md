anatomy
=======

Question and Answer Website for UTHSCSA Anatomy Class

The files for this website are provided as-is. I created it about two years ago, hence why the code 
is not in the best shape. A SQL dump is included sans the original user data. The questions and answers 
were ripped from an Excel spreadsheet using a Hex editor. It's from the Galen question bank used by numerous 
medical students.
